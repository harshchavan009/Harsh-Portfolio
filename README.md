# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/harshchavan009/Harsh-Portfolio.git
```

---

## Navigate into the Project

```bash
cd Harsh-Portfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
